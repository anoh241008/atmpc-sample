# ATMPC Sample Project

## Overview
This project was developed as part of a technical exercise. It demonstrates a full-stack web application with authentication, CRUD operations, and a responsive user interface.

## Features
- User authentication
- Dashboard
- CRUD functionality
- Responsive design
- Form validation
- Database integration
- Real-time chat between tenant and landlord (Webhook)
- Password reset (Gmail)
- Billing
- Payment integration (removed from this repository due to project owner concerns)

## Tech Stack
- Frontend: React.js
- Backend: Java (Spring Boot)
- Database: MySQL
- CSS: Tailwind CSS

## Notes
- This repository is intended as a showcase of the completed project.
- Sensitive information such as API keys, credentials, and environment-specific configuration has been removed.
- The project is still undergoing code cleanup and refactoring. Some parts of the codebase may not yet reflect the final coding style, but functionality remains intact. Improvements and optimizations are being made continuously.
